# Nexcent — Landing Page

Projeto de landing page desenvolvido como atividade prática da **UC2** do curso SenacTec.

## Visão Geral

Recriação da landing page da **Nexcent**, uma plataforma fictícia de gerenciamento de membros e comunidades. O layout foi construído a partir de um design de referência, com foco em estrutura HTML semântica e estilização com CSS puro.

## Estrutura do Projeto

```text
landing-page-nexcent/
├── index.html       # Página principal
├── css/
│   └── style.css    # Estilos da aplicação
└── img/             # Assets de imagem (SVG, ICO)
```

## Seções da Página

- **Header / Hero** — Navegação e banner principal com CTA
- **Our Clients** — Logos de clientes parceiros
- **Membership** — Cards de tipos de associação suportados
- **Pixelgrade** — Seção institucional com imagem e texto
- **Stats** — Números de impacto (membros, clubes, reservas e pagamentos)
- **How to design** — Seção informativa com imagem lateral
- **Testimonial** — Depoimento de cliente com logos de empresas
- **Blog** — Cards de artigos recentes

## Tecnologias

- HTML5
- CSS3

## Como Visualizar

Abra o arquivo `index.html` diretamente no navegador ou utilize a extensão **Live Server** no VS Code.
